## Build Instructions
#### Note: This project doesn't do anything yet.
```bash
# Clone the repo
git clone https://github.com/cheese-chess/cheese_chess
# Enter the repo directory
cd cheese_c
# Run make to compile
make
# To run it...
make run
# To install it...
make install # you may need to run this as root - use sudo/doas as required.
```
