## Dependencies
```
make
```

## Build Instructions

```bash
# Clone the repo
https://github.com/aa2006/peregrine-chess
# Enter the repo directory
cd peregrine-chess
# Run make to compile
make
# To run it...
make run
# To install it...
make install # you may need to run this as root - use sudo/doas as required.
```
